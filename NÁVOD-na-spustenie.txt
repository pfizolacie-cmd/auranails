AURA NAILS MF — webová stránka
================================

Ďakujem! V tomto priečinku je kompletný web pripravený na spustenie.

────────────────────────────────────────
AKO WEB SPUSTIŤ NAŽIVO (zadarmo, ~1 minúta)
────────────────────────────────────────
1. Choď na stránku:  https://app.netlify.com/drop
2. Tento CELÝ priečinok (rozbalený zo ZIP) pretiahni myšou do okna prehliadača.
3. Hotovo — Netlify ti dá adresu typu  https://nazov.netlify.app

────────────────────────────────────────
ALEBO NA GITHUB + GITHUB PAGES
────────────────────────────────────────
1. github.com → New repository (napr. "aura-nails") → Create
2. "Add file" → "Upload files" → pretiahni OBSAH tohto priečinka
   (index.html, všetky .jsx, priečinok assets) — index.html musí byť v koreni.
3. Commit changes.
4. Settings → Pages → Branch: main, /(root) → Save.
   GitHub ti dá adresu https://tvojmeno.github.io/aura-nails
