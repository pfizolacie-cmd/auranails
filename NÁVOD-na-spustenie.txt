AURA NAILS MF — webová stránka
================================

V tomto priečinku je kompletný web pripravený na spustenie.

────────────────────────────────────────
SPUSTIŤ NAŽIVO (zadarmo, ~1 min) — NETLIFY
────────────────────────────────────────
1. Choď na:  https://app.netlify.com/drop
2. Tento CELÝ priečinok (rozbalený zo ZIP) pretiahni myšou do okna.
3. Hotovo — dostaneš adresu typu  https://nazov.netlify.app

────────────────────────────────────────
ALEBO GITHUB + GITHUB PAGES
────────────────────────────────────────
1. github.com → New repository (napr. "aura-nails") → Create
2. "Add file" → "Upload files" → pretiahni OBSAH priečinka
   (index.html, app.bundle.js, image-slot.js, priečinok assets)
   — index.html musí byť v koreni.
3. Commit changes.
4. Settings → Pages → Branch: main, /(root) → Save.
